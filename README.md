# statezero

Small, simple, functional JavaScript library for managing immutable state.

## Installation

```
npm install statezero --save
```

Include the statezero bundle that matches your environment:

```
ls -1 node_modules/statezero/dist/
statezero.cjs.js
statezero.esm.js
statezero.umd.js
```
